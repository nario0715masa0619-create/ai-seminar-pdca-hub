# Agent Design

## エージェントの役割

### 1. Acquisition (集客エージェント)
- **役割**: ターゲット層に合わせてX広告とLPのドラフトを作成・改善する。
- **読むシート**: 親シート、X広告テンプレ、LPテンプレ
- **生成するアウトプット**: X広告文案、LPの構成・コピー案

### 2. Seminar Content (コンテンツエージェント)
- **役割**: セミナー自体の価値を高めるため、トークスクリプトとスライド資料を構築・改善する。
- **読むシート**: 親シート、LPテンプレ（訴求内容のすり合わせ）、トークスクリプトテンプレ、スライドテンプレ
- **生成するアウトプット**: トークスクリプト案、スライド構成案

### 3. Diagnostics (診断・分析エージェント)
- **役割**: 実施したセミナーや広告のパフォーマンス結果を分析し、改善案を提示する。
- **読むシート**: 各種実績データ、アンケート結果
- **生成するアウトプット**: 改善レポート、次回へのアクションプラン

### 4. Follow-up (フォローエージェント)
- **役割**: セミナー参加後のフォローアップ連絡を最適化し、商談化率を高める。
- **読むシート**: 親シート、終了後フォローテンプレ
- **生成するアウトプット**: お礼メール文面、フォローアップシナリオ
